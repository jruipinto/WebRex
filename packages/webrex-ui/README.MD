# WebRex UI
This project is the UI of "WebRex"

## How to use?
1. Build this project (if not built yet) from this folder
   - `npm run build`

2. Run WebRex from the root of the monorepo
   - `npm run dev-server:start`

3. In browser open "/webrex-ui" in the address where dev server is listening
   -  example: `localhost:3001/webrex-ui`
